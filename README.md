## Instalacion local

**Requisitos:** Node.js y npm.

1. Instalar dependencias:

```bash
npm install
```

2. Crear un archivo `.env.local` basado en `.env.example`.

```env
GEMINI_API_KEY="tu_api_key"
APP_URL="http://localhost:3000"
```

La variable `GEMINI_API_KEY` permite usar el chatbot con Gemini. Si no esta configurada, el proyecto responde con un fallback local.

3. Ejecutar el entorno de desarrollo:

```bash
npm run dev
```

4. Abrir la aplicacion en:

```text
http://localhost:3000
```

## Scripts disponibles

- `npm run dev`: inicia el servidor de desarrollo.
- `npm run build`: genera la build de produccion.
- `npm run start`: ejecuta la build de produccion.
- `npm run lint`: corre el linter configurado.
